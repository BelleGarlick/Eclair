### .removeItem
Remove an item by value.
<br/>**args**:
- value: Remove a value with a given value.
```javascript
eclair.RadioButtons()
    .add("Apple")
    .add("Banana")
    .remove("Apple")
```
### .removeIndex
Remove an item at an index.
<br/>**args**:
- index: Remove an item at an index.
```javascript
eclair.RadioButtons()
    .add("Apple")
    .add("Banana")
    .removeIndex(0)
```
